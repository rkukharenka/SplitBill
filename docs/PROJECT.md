# SplitBill — Концепция и решения

## Концепция

Telegram-бот для разделения счёта в ресторане с Mini App.

### Пользовательский сценарий

1. Пользователь скидывает боту фото чека
2. Claude Vision API (Haiku 4.5) распознаёт позиции и цены
3. Бот отправляет ссылку на Mini App
4. В Mini App каждый участник тапает на свои блюда
5. Приложение считает кто сколько должен (с учётом чаевых, общих позиций)
6. Генерируется текст для ручного перевода по реквизитам

---

## Принятые решения

### Деплой
- **Платформа:** VPS Servitro
- **Proxy:** nginx reverse proxy
- **SSL:** Let's Encrypt
- **Webhook:** фиксированный IP, HTTPS endpoint

### Оплата
- **Тип:** Реквизиты вручную
- **Механика:** пользователь вводит номер карты/телефона в профиле, бот генерирует текст для перевода
- **Пример:** "Переведи 450₽ Ивану на +7 (900) 123-45-67 (Сбер)"
- **Без интеграции** платёжных шлюзов

### Язык
- Только **русский**
- Без i18n-слоя

### Аутентификация участников
- **Гибридная схема:**
  - Telegram-пользователи — аутентификация через `initData` (автоматически)
  - Гости — вводят имя вручную при открытии Mini App
- Модель участника: `Participant(telegramId?: Long, guestName?: String)`

### Лимиты
- Максимум **20 участников** в одном разделении
- Максимум **100 позиций** в чеке

### Хранение данных
- Сессия живёт **48 часов** с момента создания
- Автоудаление через scheduled job
- Истории разделений нет

### Уведомления
- **Включены**
- Инициатор нажимает "Напомнить" → бот пингует Telegram-участников, которые ещё не выбрали блюда
- Гости без Telegram не получают уведомлений (нет `chatId`)

### Чаевые
- Быстрые кнопки: **0% / 10% / 15% / 20%**
- Поле произвольного ввода (любой процент)

### Валюта
- **Мультивалютность**
- Claude определяет валюту из чека
- Хранение: ISO 4217 код (RUB, USD, EUR и т.д.)
- Mini App отображает символ валюты

### Дополнительные фичи
- **Ручное добавление позиций** — если AI что-то пропустил
- **Разделение позиции** — одно блюдо выбирают несколько участников, стоимость делится поровну
- **Экспорт PDF** — итоги разделения в PDF для скриншота/архива
